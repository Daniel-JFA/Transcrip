# Frontend (Angular)

Angular workspace ready and connected to the transcription feature architecture:

- `src/app/core` -> singleton services + models
- `src/app/features/transcription` -> feature components/routes
- `src/app/shared/ui` -> reusable UI
- `src/styles/_theme.scss` -> CSS variables/theme

## Run

```bash
cd transcript-pro/frontend
npm install
npm start
```

Angular dev server: `http://localhost:4200`
